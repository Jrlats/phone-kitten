# phone-kitten

Angular [phonecat tutorial](https://github.com/angular/angular-phonecat) re-written
with the work-in-progress [New Angular Router](https://github.com/angular/router).


## Running the App

```shell
npm install
npm start
```

Then visit `http://localhost:8000`
